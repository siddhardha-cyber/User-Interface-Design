About Us

        Welcome to our mini website! We are passionate about web development and 
        creating simple, user‑friendly designs. This project demonstrates how 
        navigation links and iframes can be used to build a basic multi‑page site 
        without reloading the entire page.
    
        Our goal is to help beginners learn HTML, CSS, and the fundamentals of 
        frontend development step by step. As you explore, you’ll see how each 
        section—Home, About, and Contact—loads seamlessly inside the same page.
